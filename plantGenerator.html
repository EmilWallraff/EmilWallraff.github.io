Plant Generator

Plugin
Unity Engine

Plant Generator is a Unity Plugin developed by Jan Ruppert and me in cooperation with CritCrew. Its purpose is to create and grow 3D models of plants directly in Unity. Additionally, the Plant Generator can combine features of existing plants to create new ones.

The plugin generates plants by creating a Lindenmayer system based on the parameters set by the user. Additionally, the plugin can visualize plant growth in discrete steps by increasing the iterations of the Lindenmayer system.

The generated plants can consist of a trunk, branches, leaves, blossoms, and spines. The default assets for leaves, blossoms, and spines can be replaced with 3D meshes imported by the user. The plugin can also generate multiple plants at once and randomize them to have a different appearance based on user input.

The plugin also enables users to save their plant blueprints by generating a genotype file. These genotype files contain all input parameters and custom meshes, which can be loaded into the Plant Generator to modify or generate the saved plants. Moreover, the plugin allows users to merge two genotypes, with the user having the ability to determine how specific parameters should be merged.
