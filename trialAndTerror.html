Trial and Terror

2D platformer
C++, OpenGL, CMake

Trial and Terror is a 2D platformer that I programmed using my own engine, written in C++ with CMake and OpenGL. The game is centered around the main character's ability to switch between alternate versions of the game world.

The project began with the development of an engine using C++, CMake, and OpenGL, along with some small libraries for tasks such as file loading. The engine developed for the game includes many features one would expect from any game engine: memory management, sprite loading, rendering and animations, entities and components, a user interface system, level management, and more. In addition to these general features, the engine was specifically designed to efficiently manage the alternate versions of Trial and Terror's game world without wasting resources.

Trial and Terror is a mechanically tight 2D platformer that offers precise jumping and dashing mechanics. The player is also provided with the unique ability to switch between two alternate versions of the game world: a red version and a blue version. While these versions share the same base layout, they feature different details. Depending on the level, this ability creates either an engaging, fast-paced platforming experience or an intriguing puzzle challenge.
